# Excel Data Analysis Project

## Overview

This project was developed using Microsoft Excel as part of an Excel training assignment. The goal of the project is to demonstrate data organization, formula usage, data analysis, and visualization techniques in Excel.

## Features

* Data cleaning and formatting
* Name and surname separation from full names
* Text manipulation formulas
* Employee working year calculations
* Salary increase calculations
* Percentage change calculations
* IF formula usage for performance evaluation
* COUNT and COUNTIF functions for statistical summaries
* VLOOKUP function for data retrieval
* Pivot Table analysis
* Pivot Chart visualization
* Slicer-based filtering

## Project Structure

### Employee Data Sheet

Contains employee information including:

* Full Name
* First Name
* Last Name
* Employee ID
* Department
* Hire Date
* Working Years
* Salary Information
* Performance Score
* Status Evaluation

### VLOOKUP Sheet

Demonstrates the use of the VLOOKUP function for retrieving department information based on employee IDs.

### Pivot Analysis Sheet

Includes:

* Pivot Table reports
* Department-based performance analysis
* Interactive slicer filtering
* Performance visualization chart
